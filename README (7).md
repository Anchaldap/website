## 🚀 How to Run

No installation needed. Just open the file in any browser:

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Open in browser
open undergrads-media.html
```

Or deploy directly to:
- **GitHub Pages** — Enable in repo Settings → Pages → select main branch
- **Netlify** — Drag and drop the HTML file
- **Vercel** — Import repo and deploy
